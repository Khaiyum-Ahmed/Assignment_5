Important FAQ

What is the Document Object Model (DOM)?
The DOM (Document Object Model) is a tree-like structure the browser creates from an HTML page. It represents every element as an object, allowing JavaScript to access and change content, styles, and structure dynamically. This is what makes webpages interactive without needing to reload the entire page.
How do you select an element from the DOM?
You can select elements using JavaScript methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll. These methods let you target elements by their ID, class, tag name, or CSS selector. Once selected, you can read or modify the content, style, or behavior of those elements.
What is event delegation in the context of the DOM, and why is it useful?
Event delegation is a technique where a parent element handles events for its child elements using a single event listener. Instead of adding listeners to each child, you add one to the parent and use event.target to identify which child triggered the event. It's useful for:
  1. Better performance
  2. Fewer event listeners
  3. Handling dynamically added elements
Example use: click events in a list.
How do you manipulate an elements attributes and style using the DOM?
You can manipulate an element's attributes using methods like setAttribute, getAttribute, and removeAttribute. To change styles, use the element’s style property, such as style.color or style.fontSize. These allow you to dynamically update how an element looks or behaves, making your webpage interactive and responsive to user actions.
